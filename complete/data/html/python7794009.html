#6

** can also be used as a function parameter to pass a variable number of keyword arguments to a function. Example:

def x(**kwargs):
    for key, value in kwargs.items():
        print "%s: %s" % (key, value)

x(y=2, z=3, a=1, b=2)

Prints:

y: 2
z: 3
a: 1
b: 2
