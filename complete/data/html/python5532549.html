I played a bit with python in Notepad++ and I had a problem with the tab. Since in python you indentation is really important notepad ++ put space instead of a tab.

So to change that you can go in notepad ++ preference -> select "language menu tab and setting" tab Then select python in the "tab setting" box and select replace tab by space.
