Merging items in a list - Python

Say I have a list in python, like such:

list=[1,2,3,4,5]

How would I merge the list so that it becomes:

list= [12345]

If anyone has a way to do this, it would be greatly appreciated!!

#1
>>> list=[1,2,3,4,5]
>>> k = [str(x) for x in list]
>>> k
['1', '2', '3', '4', '5']
>>> "".join(k)
'12345'
>>> ["".join(k)]
['12345']
>>> 
>>> [int("".join(k))]
[12345]
>>>
