Letter Count Dictionary

My homework problem is to write a Python function called LetterCount ( ) which takes a string as an argument and returns a dictionary of letter counts. However , my code includes white space and commas as a part of the dictionary which i do n't want . Can you please help me out. How do i remove the white space from my list ? Here is my code ? 

import string
def LetterCount(str):
    str= str.lower().strip()
    str = str.strip(string.punctuation)
    list1=list(str)
    lcDict= {}
    for l in list1:
        if l in lcDict:
            lcDict[l] +=1
        else:
            lcDict[l]= 1
    print lcDict

LetterCount("Abracadabra, Monsignor")

string.strip ( s [ , chars ] ) Return a copy of the string with leading and trailing characters removed. If chars is omitted or None , whitespace characters are removed. If given and not None , chars must be a string ; the characters in the string will be stripped from the both ends of the string this method is called on . Changed in version 2.2.3 : The chars parameter was added. The chars parameter can not be passed in earlier 2.2 versions .

# 5

Extending python - Letter Count Dict : 

from collections import Counter

def LetterCount(text):
    return Counter(c for c in text.lower() if c.isalpha())
