What would randrange be in Java ?

You can get a random integer in a specific range from Java 's Random class by calling the nextInt ( int n ) method. For example, 

Random rand = new Random();
int a = rand.nextInt(7);

will give you a random integer > ; = 0 and < ; 7. This is n't exactly the same as randrange in Python , but you could use it as the index to an array of objects , or as the value of a roll of a single die .
